# photo-presen-slide

## Setup
### node.js install
```
$ brew install node.js
```

### Grunt install
```
$ npm install -g grunt-cli
```
### Git Clone
```
$ git clone https://github.com/alexandritex2007/photo-presen-slide.git
```
### Move to directory
```
$ cd photo-presen-slide
```
### Installing packages
```
$ npm install
```
## Start-up
```
$ grunt serve
```
[http://localhost:8000](http://localhost:8000)
